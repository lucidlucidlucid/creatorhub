About GTAG Colors

Gorilla Tag uses a simplified color system where each RGB channel ranges from 0-9 instead of the standard 0-255. This converter helps you translate between these formats.

Use the sliders or input values directly to see how GTAG colors map to standard web colors. Click on the HEX or RGB values to copy them to your clipboard.
